Design notes
  • Inspiration: Google Now
  • The purpose of the home page is to bring visibility to things that are most important to the user.
  • Pinned reports
    • Users could have the ability to add/pin a status dashboard to the homepage
    • The option to pin a dashboard is located in the nav bar dropdown menu
    • Any edits made to a dashboard report would be reflected in the homepage totals
    • Users have the ability to add Analytics reports to the dashboard. Example: see the Google Analytics Add to dashboard feature.
    • If a user has bookmarked filters on an exp.o Analytics report, the report on the homepage would refect those settings
  • Starred: Panel would display automatically when the user has starred at least one thing (shipment, container, order, etc)
